    Pursuing Raspberry Pi

    After a long research on Raspberry Pi, I’ve made the decision of getting myself the B+. Now, the problem arised of buying the product here in India which was made by the company ELEMENT14. It was quite difficult to get it directly via ordering it from their website to my place located in India. After lot of difficulties I made my way on to purchasing it via eBay India.

    It was very exciting to know more about the Raspberry Pi as it opened the doors to endless possibilities by building such a small computer which is actually the size of a credit card.

    Prerequisites

    • The Raspberry Pi B+ itself, ofcourse you need that.
    • An Internet connection with an ethernet cable.
    • A keyboard (optional: + mouse) for interacting with your Pi.
    • A HDMI enabled monitor/television.

    Getting Started

    There are multiple ways of getting started with Raspberry Pi, by installing operating systems listed on their website. Its of great help to check those install guides as they guide you the best for installation.

    After some good amount of research, I had made my mind on going with the newbie operating system called NOOBS.

    First Step

    • Insert the SD Card which is 4GB or greater. A SD card acts like a harddisk on the Pi.

    Second Step

    Formatting of SD Card. Below are the instructions for operating system specific installing NOOBS on your SD card.

    On Windows

    • Download the SD Association’s Formatting Tool from https://www.sdcard.org/downloads/formatter_4/eula_windows/
    • Install and run the Formatting Tool on your machine
    • Set FORMAT SIZE ADJUSTMENT option to ON in the Options menu
    • Check that the SD card you inserted matches the one selected by the Tool
    • Click the Format button

    On Mac

    • Download the SD Association’s Formatting Tool from https://www.sdcard.org/downloads/formatter_4/eula_mac/
    • Install and run the Formatting Tool on your machine
    • Select Overwrite Format
    • Check that the SD card you inserted matches the one selected by the Tool
    • Click the Format button

    On Linux

    • We recommend using gparted (or the command line version parted)
    • Format the entire disk as FAT

    [Install instructions provided under the README of NOOBS Operating System]

    Third Step

    • After successful completion on the formatting of the SD Card, the installation of the operating system has to be done.
    • Download NOOBS Operating System from their website.
    • Copy the extracted files onto the SD card that you just formatted so that this file is at the root directory of the SD card.

    Please note that in some cases it may extract the files into a folder, if this is the case then please copy across the files from inside the folder rather than the folder itself.

    Fourth Step

    • Insert the SD Card into the Pi and connect the Power Supply.
    • Pi will now boot into NOOBS and should display a list of operating systems that you can choose to install.
    • Choose Raspbian.

    If your display remains blank, you should select the correct output mode for your display by pressing one of the following number keys on your keyboard: - HDMI mode - this is the default display mode. - HDMI safe mode - select this mode if you are using the HDMI connector and cannot see anything on screen when the Pi has booted. - Composite PAL mode - select either this mode or composite NTSC mode if you are using the composite RCA video connector. - Composite NTSC mode
